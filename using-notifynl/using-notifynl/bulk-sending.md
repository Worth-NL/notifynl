# Bulk sending

You can send a batch of up to 100,000 messages at once.

There’s a [daily limit](daily-limits.md) to the number of messages you can send.

To send a batch of messages:

1. Go to the Templates page.
2. Add a new template or choose an existing template and select Get ready to send.
3. Select Upload a list of email addresses, Upload a list of phone numbers, or Upload a list of addresses.

Find out [how to personalise your messages](personalisation.md).

You can also [schedule a batch of emails and text messages](schedule-messages.md).
