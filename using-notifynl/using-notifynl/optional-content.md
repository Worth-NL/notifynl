# Optional content

Add optional content to a template to tailor your messages for different audiences.

For example, if you only want to show a sentence to people who are under 18:

* put it inside double brackets
* give it a unique placeholder name, followed by 2 question marks

Your optional content should look like this:

```
((under18??Please get your application signed by a parent or guardian.))
```

You cannot add a placeholder for personalisation within optional content.

For each person you send the message to, specify ‘yes’ or ‘no’ to show or hide the optional content.

You can either:

* do this yourself when you send the message
* upload a list of personal details and let Notify do it for you

If you upload a list of personal details, the column names need to match the placeholder names in your template.
