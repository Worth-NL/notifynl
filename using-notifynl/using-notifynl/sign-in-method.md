# Sign-in method

When you sign in to NotifyNL, you’ll need to enter:

* your email address
* your password
* a text message code that NotifyNL sends to your phone

If signing in with a text message is a problem for your team, you can also sign in using a link that’s sent in an email.

For security, you should only do this if you use:

* a work-managed device
* a work email address protected by multi-factor authentication (MFA)

### How to change the sign-in method for your team

1. Go to the Settings page.
2. Select Change on the Sign-in method row.

Now go to the Team members page to choose the sign-in method for each person on your team.
