# Text message sender ID

The text message sender ID tells your users who the message is from.

To change the text message sender ID from the default of ‘NOTIFYNL’:

1. Go to the Text message settings section of the Settings page.
2. Select Manage on the Text message sender IDs row.
3. Select Change or Add text message sender ID.

If you want to [receive text messages](receive-text-messages.md), your sender ID will be an 11-digit phone number.
